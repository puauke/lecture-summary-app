## 🔒 セキュリティ対策実装済み

このアプリにはセキュリティ対策が実装されています。詳細は [SECURITY.md](SECURITY.md) をご参照ください。

### 実装済みセキュリティ機能

- ✅ **ファイルアップロード検証** - 拡張子・サイズチェック、ファイル名サニタイゼーション
- ✅ **SSRF 対策** - ローカルホスト・プライベート IP へのアクセス禁止
- ✅ **入力値検証** - 検索クエリ・URL の安全性確認
- ✅ **タイムアウト設定** - 10秒で無応答なリクエストは自動終了
- ✅ **コンテンツサイズ制限** - 2MB 以上のコンテンツは制限
- ✅ **バージョンロック** - 依存ライブラリの脆弱性対策
- ✅ **.gitignore 設定** - API キー・機密情報の保護

## 🚀 クイックスタート

### 1. 環境セットアップ

```bash
# ディレクトリに移動
cd lecture_summary_app

# 依存ライブラリをインストール
pip install -r requirements.txt
```

### 2. 環境変数の設定

`.env` ファイルを作成：

```
GOOGLE_API_KEY=your_api_key_here
```

⚠️ **注意**: `.env` ファイルは **絶対に GitHub にアップロードしないでください！**

### 3. アプリを起動

```bash
streamlit run app.py
```

スマホでアクセス（同じ Wi-Fi）：

```bash
streamlit run app.py --server.port 8501 --server.address 0.0.0.0
```

PC の IP を確認：

```powershell
ipconfig
```

スマホのブラウザ：

```
http://<PCのIP>:8501
```

## 📋 機能一覧

### 📂 カテゴリ管理
- 既存のカテゴリから選択
- 新しいカテゴリを作成
- カテゴリごとにデータを保存

### 📥 データ取り込み
- **ファイル**: PDF / TXT のみ（5MB まで）
- **Web 検索**: キーワードで検索
- **URL 直接入力**: 特定ページを指定
- **RSS フィード**: 最新記事を自動取得

### 📋 出力機能
- **全体まとめ**: 最重要ポイント・流れ・実践応用
- **統合要約**: 詳細な統合ノート形式
- **関連資料**: AI 推薦の参考文献
- **Q&A**: AIチューター機能
